### 工欲善其事，必先利其器

Android Studio `常用`快捷键 (for Mac)，亲测好用。
 
键 | 说明 |
-|-
⌥ | option / alt
⇧ | shift
⌃ | control
⌘ | command

查找相关

快捷键 | 功能 |  
-|-
⌘O | 全局查找 class 类名 | 
⌘F | 在当前编辑文件中查找 |
⌘F12 | 当前编辑的文件中结构快速导航 |
⌘F7 | 列出全局项目中该类／方法 被引用／调用的情况 |
⌘E | 打开最近访问的文件列表 |
⇧⌘O | 全局项目查找文件 |
⌘P | 列出函数方法一系列的有效参数，在调用方法的时候 |
F1 | 查看类，方法的注释文档 |

控制操作相关

快捷键 | 功能 |  
-|-
⇧⌥up/down | 移动行上下移动 |
⌘d | 复制行 |
⌘J | 快速生成模版代码块，如if,while,return, 单例，const等
⌘N | 快速生成getter／setter方法，构造方法，toString()方法等
⌥Enter | 项目快速修复（导包）
⌥⌘L | 代码格式化
⌘⇧U | 切换选中内容的大小写，命名全局常量时比较有用
